## Installation

```bash
npx changeelog/reactuse@latest add useTimestamp
```

## Usage

```tsx
import { useTimestamp } from '#';

const Component = () => {
  // Basic usage
  const timestamp = useTimestamp();

  // With controls
  const {
    timestamp: controlledTimestamp,
    pause,
    resume,
  } = useTimestamp({ controls: true });

  return (
    <div>
      <p>Timestamp: {timestamp}</p>
      <p>Controlled Timestamp: {controlledTimestamp}</p>
      <button onClick={pause}>Pause</button>
      <button onClick={resume}>Resume</button>
    </div>
  );
};
```

## Reference

### useTimestamp

The `useTimestamp` hook provides the current timestamp and optionally exposes controls for pausing and resuming updates.

### Return Value

If `controls` is `false`, the hook returns the current `timestamp` as a `number`.

If `controls` is `true`, the hook returna an object containing the following properties:
